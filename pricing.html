Configuration prête pour Firebase

Cet aperçu garde la tarification sous forme de configuration structurée, afin que le paiement Stripe puisse être activé plus tard en changeant les données et les feature flags, sans refonte de la page.

Document Firestore

Chemin recommandé pour la production : publicConfig/pricing. Il peut être lisible par tous les utilisateurs et modifiable uniquement par les administrateurs.


        

        

          
Future bascule Stripe

          
L'achat est désactivé pour le moment. Plus tard, activez billingEnabled et stripeEnabled, ajoutez les identifiants de prix Stripe, puis connectez le paiement.

          
{
  "billingEnabled": false,
  "stripeEnabled": false,
  "checkoutMode": "disabled",
  "purchaseButtons": "disabled"
}
